# LemiCraft

LemiCraft is a Minecraft project with its own launcher, website, wiki and public API

## Projects

### [LemiCraft-Launcher](https://github.com/LemiCraft/LemiCraft-Launcher)
A custom Windows launcher written in C# and WPF. Supports Microsoft and
Ely.by authentication, automatic modpack installation and updates, server
status monitoring, crash analysis, and skin management

### [LemiCraft-Site](https://github.com/LemiCraft/LemiCraft-Site)
The main website

### [LemiCraft-Wiki](https://github.com/LemiCraft/LemiCraft-Wiki)
The official player wiki

### [LemiCraft-API](https://github.com/LemiCraft/LemiCraft-API)
Public API for accessing server data - player info, statistics, and more
Documentation available at [lemicraft.ru/api-reference](https://lemicraft.ru/api-reference)
